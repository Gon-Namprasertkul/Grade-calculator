# Grade Calculator

requirement:

1.Command line or terminal

2.Python 3.x

This calculator will help you find your grade

How to use(Windows):

1.Open .bat file with command line or terminal

2.Enter your score

How to use(macOS):

1.Go to mac folder

2.select AF.command

3.open with terminal

if not work(mac):

1.go to terminal

2.type chmod +x (Drag the .command file here)

3.press enter
